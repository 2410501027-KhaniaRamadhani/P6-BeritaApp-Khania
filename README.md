Fitur yang sudah berhasil 
Pertama, bookmark atau menyimpan artikel: Aplikasi bisa menyimpan artikel favorit pengguna. Data tersimpan langsung di penyimpanan HP, jadi meskipun sedang offline atau tidak ada internet, artikel yang sudah di-bookmark tetap bisa dibuka dan dibaca. Cukup tekan icon bookmark, maka artikel akan tersimpan.
Kedua, pencarian dengan debounce: Fitur pencarian dibuat tidak langsung mencari setiap kali huruf ditekan. Aplikasi akan menunggu sekitar setengah detik setelah pengguna selesai mengetik baru melakukan pencarian. Ini berguna agar kuota API tidak cepat habis dan aplikasi tetap berjalan lancar.
Ketiga, filter berita: Pengguna bisa memfilter berita berdasarkan sumber tertentu seperti CNN atau BBC. Selain itu, bisa juga filter berdasarkan rentang tanggal. Contohnya mencari berita tentang sepak bola dari sumber tertentu antara tanggal 1 sampai 10 Januari.
Keempat, mode gelap atau dark mode: Tampilan aplikasi bisa mengikuti pengaturan gelap terang dari HP. Kalau HP sedang mode gelap, aplikasi otomatis berubah menjadi gelap. Ini enak dipakai saat malam hari atau di ruangan redup.
Kelima,  bagikan artikel: Pengguna bisa membagikan artikel ke teman melalui aplikasi pesan seperti WhatsApp atau Telegram. Cukup tekan icon share, maka link dan judul artikel akan langsung terbawa.

Kendala yang saya hadapi adalah Awalnya saya kesulitan mendapatkan API Key karena harus mendaftar di website NewsAPI dan verifikasi email. Sempat juga terjadi error karena ada package yang belum terinstall, seperti datetimepicker. Selain itu, halaman beranda sempat kosong karena proses pengambilan data belum berjalan otomatis.

Hasil Akhir: Aplikasi BeritaApp sekarang sudah berjalan dengan baik. Bisa menampilkan berita dari berbagai kategori seperti umum, teknologi, olahraga, bisnis, dan kesehatan. Bisa mencari berita dengan filter sumber dan tanggal. Bisa menyimpan artikel favorit. Bisa berganti tema antara gelap dan terang. Bisa membagikan artikel ke teman.
